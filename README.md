terminal-webcam
===============

A webcam that can be run on a bare terminal.

setup
-----

Tested on Ubuntu 12.04

```python
sudo apt-get install python-opencv libncurses5-dev libncurses5 ncurses-term
```

usage
-----

```python
python capture.py

# For colored version which may be slower, needs optimizing
python color.py
```
